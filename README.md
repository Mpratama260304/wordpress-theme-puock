## Installation

Download the latest version from [Releases](https://github.com/Licoy/wordpress-theme-puock/releases), then go to WordPress admin panel → Appearance → Themes → Add New → Upload Theme, select the Puock theme package and install.

**Note: To prevent theme incompatibility, please backup your data before installing the theme.**

**Important: Please do not clone or download the repository directly for use. Download from the releases page instead.**

### Requirements

- WordPress `>=6.0`
- PHP `>=7.4`

## Localization

This theme supports multiple languages:
- **English (en_US)** - Default language
- **Indonesian (id_ID)** - Full translation
- **Traditional Chinese (zh_TW)** - Full translation

To change the language, go to WordPress admin → Settings → General → Site Language.
